# Travel-agency
A responsive travel agency website template featuring destination packages, a search interface, and a modern, user-friendly design built with HTML, CSS, and JavaScript.

# Travel Agency Website

This is a modern, responsive landing page for a travel agency, designed to help users search for destinations and view popular travel packages.

## 🚀 Features
- **Responsive Design:** Looks great on mobile, tablet, and desktop.
- **Hero Section:** Engaging landing area with a search functionality.
- **Package Cards:** Organized display of travel deals with "Book Now" options.
- **Interactive:** Simple JavaScript alerts for search and booking interactions.

## 🛠️ Technologies Used
- **HTML5:** Structuring the content.
- **CSS3:** Styling and layout (Flexbox).
- **JavaScript:** Basic interactivity.

## 💻 How to View
1. Clone this repository to your local machine.
2. Open `index.html` in any web browser to view the site.
3. *Alternatively, check the live demo hosted on GitHub Pages.*
